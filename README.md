# Responsive Fitness Landing Page

This project showcases a visually appealing and user-friendly landing page tailored for fitness brands and enthusiasts. It is crafted using HTML and CSS, ensuring a seamless experience on any device.

## Live Demo

Check out the live demo [here](https://dianalit13.github.io/fitness/).

## Features

- **Fully Responsive Design**: Adapts to various screen sizes, providing an optimal viewing experience on mobile, tablet, and desktop.
- **Well-Structured Sections**: Includes sections for services, testimonials, and contact information.

## Technologies Used

- **HTML**: The backbone of the webpage, creating the structure and content.
- **CSS**: Styles the webpage for a polished and professional look.
- **Responsive Layouts**: Utilizes Flexbox and CSS Grid for an adaptable design.

## How to Use

To get started with the project:

1. Clone the repository:
   ```bash
   git clone https://github.com/dianalit13/fitness.git
   ```

2. Navigate to the project directory:
   ```bash
   cd fitness
   ```

3. Open `index.html` in your favorite web browser.
